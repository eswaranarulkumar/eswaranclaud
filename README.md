# eswaranclaud

Personlig nettside for [eswaran.no](https://www.eswaran.no) — kunnskap om hinduisme og vedisk kultur på norsk.

## Innhold

- `index.html` — hele nettsiden (HTML + CSS i én fil, ingen byggetrinn nødvendig)

## Kjør lokalt

Åpne `index.html` i nettleseren, eller start en enkel server:

```bash
python3 -m http.server 8000
```

## Publisering

Siden kan publiseres gratis med GitHub Pages: Settings → Pages → velg branch `main` og mappe `/ (root)`.
